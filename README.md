# 🎮 Mario Platformer V1

A simple 2D Mario-style platformer game built with HTML, CSS, and JavaScript.  
Built for desktop and mobile browsers using Canvas.

---

## 🚀 Features

- 🧑 Smooth player movement (keyboard + touch controls)
- 🌍 Scrolling world with camera follow
- 🪙 Coins collection + score system
- 👾 Enemy AI movement
- 🏁 Win flag / level completion system
- 📱 Mobile-friendly buttons
- 🎨 Optional sprite/image support (PNG assets)
- ⚡ Lightweight and fast (no frameworks)

---

## 🎮 Controls

### 💻 Keyboard
- ⬅ Arrow Left → Move left  
- ➡ Arrow Right → Move right  
- Space → Jump  

### 📱 Mobile
- ⬅ Button → Move left  
- ⬆ Button → Jump  
- ➡ Button → Move right

## 🚀 How to Run Locally

1. Download or clone this repository  
2. Open `index.html` in any browser  
3. Start playing instantly 🎮  

---

## 🚀 Deploy to GitHub Pages

1. Go to your repository on GitHub  
2. Click **Settings**  
3. Go to **Pages**  
4. Under Source:
   - Branch: `main`
   - Folder: `/root`
5. Click Save  
6. Your game will be live here:https://emmyjosh65.github.io/mario-platformer----
